Notes on Release 1.5 of ZETA

Wolfgang Grieskamp

This release supports the core functionality of the ZETA system - type setting, type checking and executing Z specifications. In comparsion to the release 1.03 of November '98, the following changes have been made:

  • ZETA now runs under Java 1.2 and upwards (and only Java 1.2)
  • numerous bug-fixes in the Z parser and type-checker
  • numerous bug-fixes in the XEmacs and Swing GUIs
  • a new execution machine and compiler
  • system now under GPL

If you have used ZETA before, be aware of the following incompatbilities:

  • in this release, µSZ and Dynamic Z is not supported
  • the %%toolkit directive is not longer required
  • their is only one toolkit packed in this release (no unit-weighted numbers)
