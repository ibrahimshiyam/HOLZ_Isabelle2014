2. XEmacs Support for ZETA

ZETA can be operated from within a comfortable XEmacs mode, zeta-source-mode. To start a ZETA session from within XEmacs, issue M-x zeta.

The mode realizes a session log as described for the graphical user interface. The following snapshot represents the same operation state as in the previous section, where we just executed the expression BirthdayBook:

Clicking on the highlighted locaters visits the referred sources.

In the commander's buffer, you can type in ZETA commands. The set of available commands is displayed with the help command. C-up and C-down lets you step through the command history. In order to enter multiline arguments, quote them, as in 

 e "
    date \lbind day==1, month==2, year==1998 \rbind
  "

ZETA's operations can be also reached via the ZETA menu and various key-shortcuts which all start with C-q:

The current default unit (section) to operate on is defined with C-Q u. If any of the commands which operate on sections, such as type checking, execution, or formatting are activtated via the menu or by a key-shortcut from within a source buffer, ZETA tries to automatically determine the default unit by scanning for \zsection directives in the buffer.

A non-terminating operation can be interrupted using C-q C-c. This is in particular useful for execution, where non-termination is a possible behaviour.
