3. Batch Operation

On startup, the ZETA system reads and executs commands in the files $ZETABASE/lib/zetarc, $HOME/.zetarc and ./zetarc in this order. The first file usually loads the adaptors. You can use any ZETA command in the batch files, but note that some commands are not available unless the adaptors which provide them have been loader. Use the help command to get a list and short explanation of all available commands. Useful commands in zetarc-scripts are:

  • plugin <class>: Plugin an adaptor specified by the given class. The class must be found in ZETA's classpath.
  • load <filename>*: load a LATEX file.
  • path <pathname>*: set the LATEX search path.
  • make <unitname> <content-type>: establish the given content type for the unit.
