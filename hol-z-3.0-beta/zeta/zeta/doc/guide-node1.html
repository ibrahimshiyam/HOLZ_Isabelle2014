1. Introduction

The Zeta system is devided into the session system (see Sect. 2.2), the shell (see Sect. 2.3) and the adaptors (see 2.4). The session is responsible for the coordinating tasks, in particular for the computation. The shell takes care of interaction wiht the user. The actual work is done by the adaptors. The perform transformations and offer commands (forms) to execute specific tasks. Any adaptor can be plugged into the system. The adaptor tells the system what services it provides and by that augments the functionality of the hole systems. Adaptors use functionality of other adaptors in asking the session for a specific content, the sessin then looks for adaptors that are able to compute this content.
