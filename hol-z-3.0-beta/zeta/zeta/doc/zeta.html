The Operation of ZETA

Wolfgang Grieskamp

July 26, 2000

Abstract:

The ZETA tool environment provides an open framework for the integration of tools and notations. Regarding its custom functionality and semantic support, it bases on the Z notation. This document gives a brief overview on the operation of the ZETA environment, addressing users working with the environment.
